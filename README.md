# Digital-Pass
A full-stack web application designed to digitize and automate campus entry management using secure QR-based verification.
This system replaces traditional paper passes with a centralized, efficient, and secure platform for managing entry permissions of students, staff, and visitors.
Managing campus access manually leads to delays, security risks, and poor record keeping. This project provides a digital solution where users can request, approve, and verify entry passes seamlessly.

Features:
 Multi-Role Authentication
    Visitor, Officer, and Admin access control
 Digital Pass Generation
    Users can request entry passes online
 Approval Workflow
    Officers can approve/reject requests in real-time
 QR Code Verification
    Secure and quick validation at entry points
 Audit Logging
    Tracks all activities for accountability
 Real-Time Updates
    Instant status updates for users

Why this project matters:
Managing campus entry using manual passes leads to:
 Long queues and delays
 Security risks (fake/misused passes)
 Poor record tracking
This system solves these problems by introducing a smart, automated, and secure digital workflow for managing entry permissions.
Tools:
Frontend	HTML, CSS, JavaScript
Backend	Node.js + Express 
Database	MySQL
